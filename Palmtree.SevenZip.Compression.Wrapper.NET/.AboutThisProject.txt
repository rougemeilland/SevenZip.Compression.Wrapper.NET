このプロジェクトは、ユーザーに公開されるクラスを持つライブラリのプロジェクトです。
このライブラリのターゲットは「Any CPU」であり、プラットフォームに依存しません。
このライブラリの内部の処理はプラットフォームに依存する外部のネイティブコードで行われます。
このライブラリは様々なネイティブコードに対応可能です。(Windows/Linux/MaxOs, x86/x64/ARM/ARM64)
しかし、実際に実装されているネイティブコードは Windows/Linux および x86/x64 のみです。
これは、私が開発のためのハードウェアが所有していないからです。

This project is a library project with classes exposed to users.
The target of this library is "Any CPU" and is platform independent.
The internal processing of this library is performed by external native code that is platform dependent.
This library can support various native codes. (Windows/Linux/MaxOs, x86/x64/ARM/ARM64)
However, the actual native code implementation is only for Windows/Linux and x86/x64.
This is because I don't own any development hardware.
