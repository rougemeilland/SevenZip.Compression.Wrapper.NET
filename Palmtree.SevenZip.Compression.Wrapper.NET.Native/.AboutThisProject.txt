このプロジェクトは、7-zipのネイティブライブラリがサポートしている各々のインターフェースにアクセスする手段を提供するネイティブライブラリです。
このプロジェクトはプラットフォーム間で共通のソースを保持するためのプロジェクトであり、このプロジェクトの出力は公開パッケージには含まれません。

This project is a native library that provides a way to access each interface supported by the 7-zip native library.
This project is for maintaining a common source across platforms, and the output of this project is not included in the public package.
