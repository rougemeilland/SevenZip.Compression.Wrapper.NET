このプロジェクトは既に使用されていません。

当初、プラットフォームに依存するプラグインアセンブリをC++/CLIで記述するために、このプロジェクトは作成されました。
しかし、C++/CLIには以下の制限があり将来性に疑問があったため、「p/Invike」を使用した実装に切り替えられました。
・C++/CLIではx86/x64以外のアーキテクチャがサポートされない。(現在だけではなく将来にわたっても)。
・C++/CLIではWindows以外のOSでの動作がサポートされない。(現在だけではなく将来にわたっても)。


This project is no longer in use.

Initially, this project was created to write platform-dependent plugin assemblies in C++/CLI.
However, C++/CLI had the following restrictions and had doubts about its future, so it was switched to an implementation using "p/Invike".
- C++/CLI does not support architectures other than x86/x64. (Not only now but in the future).
- C++/CLI does not support operation on OS other than Windows. (Not only now but in the future).
